Get started in minutes

From concept to production in just a few simple steps

1

Install & Configure

Install our SDK with a single command and configure your project with minimal setup. Works with any Node.js, Python, or Go application.

npm install @trigger.dev/sdk
2

Define Your Jobs

Create background jobs using our intuitive API. Define triggers, schedules, and workflows with type-safe code.

client.defineJob({ id: "process-upload", trigger: webhook() })
3

Deploy & Monitor

Deploy your jobs to our global infrastructure and monitor execution in real-time with our comprehensive dashboard.
